# To-Do List Interaktif (Interactive To-Do List)

Ini adalah aplikasi To-Do List interaktif sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript murni. Aplikasi ini memungkinkan pengguna untuk menambah, menandai selesai, dan menghapus tugas.

Aplikasi ini juga dilengkapi dengan fitur penyimpanan data menggunakan `localStorage`, sehingga tugas tidak akan hilang saat browser ditutup atau di-refresh. Selain itu, daftar tugas akan otomatis menyortir tugas yang sudah selesai ke bagian bawah untuk fokus yang lebih baik.

## âœ¨ Fitur-Fitur

* **Tambah Tugas:** Menambahkan tugas baru ke dalam daftar.
* **Tandai Selesai:** Mengubah status tugas menjadi selesai (dengan efek visual teks dicoret).
* **Hapus Tugas:** Menghapus tugas secara permanen dari daftar.
* **Validasi Input:** Mencegah penambahan tugas kosong atau tugas duplikat.
* **Penyimpanan Persisten:** Semua tugas disimpan di `localStorage` browser, sehingga data tidak hilang saat halaman dimuat ulang.
* **Penyortiran Otomatis:** Tugas yang sudah selesai akan selalu dipindahkan ke bagian bawah daftar secara otomatis.

Proyek ini dibuat sepenuhnya dengan teknologi web dasar di sisi klien, tanpa menggunakan *framework* atau *library* eksternal (seperti React, Vue, atau jQuery).

* **JavaScript (ES6+):** Untuk semua logika aplikasi, termasuk:
    * Manipulasi DOM
    * Event Handling (Event Listeners, Event Delegation)
    * Mengelola `localStorage`
